#Assistente de Voz Multi-Idiomas Com Whisper e ChatGPT

Este projeto consiste em um assistente desenvolvido no Google Colab que utiliza reconhecimento de voz e inteligência artificial para transformar e melhorar textos automaticamente.

🚀 Como funciona

O funcionamento do assistente segue as seguintes etapas:

1. O usuário fala uma frase ou texto através do microfone;
2. A aplicação converte a fala em texto;
3. Esse texto é enviado para a API do ChatGPT;
4. A IA reescreve ou melhora o conteúdo;
5. O resultado é exibido na tela e pode ser salvo como print para o usuário.
